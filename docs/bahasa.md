# AKSARA

AKSARA adalah bahasa pemrograman interpreted dengan kata kunci berbahasa Indonesia.

## Menjalankan Program

```bash
aksara jalankan examples/halo.aks
```

## Contoh

```aksara
buat nama = "Reza"
buat umur = 21

jika umur >= 18
    tulis "Halo, " + nama
jika_tidak
    tulis "Anak"
selesai
```

## Tipe Data Awal

- integer
- float
- string
- boolean
- list
- dictionary
- null melalui `kosong`
- function untuk fungsi bawaan

## Fungsi Standar Awal

`panjang`, `acak`, `huruf_besar`, `huruf_kecil`, `gabung`, `pisah`, `tunggu`, `waktu`,
`tanggal`, `buka`, `tulis_berkas`, `hapus`, dan `buat_folder`.
