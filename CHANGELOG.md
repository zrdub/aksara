# Changelog

All notable changes to AKSARA are documented in this file.

This project follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and uses semantic versioning for public releases.

## Unreleased

### Added

- Repository governance documents for contributing, conduct, security reporting,
  release readiness, issue reporting, pull requests, and dependency maintenance.
- GitHub Actions workflow for Python 3.12 and Python 3.13 that installs the
  package, checks formatting, runs pytest, and builds distribution artifacts.
- Dependabot configuration for GitHub Actions and Python package dependencies.
- Expanded README with badges, installation instructions, examples,
  documentation links, roadmap, and contributing guidance.

## 0.1.0 - 2026-06-27

### Added

- Initial AKSARA language package with lexer, parser, AST, runtime,
  interpreter, standard library, CLI, examples, documentation, and tests.
